# Sander Mod - Quick Start Guide

## Installation

### Step 1: Get the DLL
Download `Sander.dll` from:
- **Linux**: `bin/publish/linux/Sander.dll`
- **Windows**: `bin/publish/windows/Sander.dll`

### Step 2: Install
1. Open your Space Station 14 mod folder
2. Place `Sander.dll` in the mods directory
3. Launch the game

### Step 3: Open Menu
Press `F1` or use the verb menu to access mod controls.

---

## Feature Overview

### 🎯 Compass (Always Active)
The compass shows nearby entities around you:

| Symbol | Meaning |
|--------|---------|
| 🔵 Dark Blue (center) | Your position |
| 🟢 Green | Crew members |
| 🟠 Orange | NPCs |
| 🟡 Yellow (▲) | Searched items |
| 🔴 Red | Syndicate |
| 🔵 Cyan | Pirate |
| 🟢 Green (dot) | Target coordinates |

### 💡 Visual Controls

| Button | Function |
|--------|----------|
| Fullbright | Removes all darkness |
| Shadows | Toggle shadow rendering |
| FOV | Enable wider field of view |

### ♿ Sound Subtitles

Enable "Sounds" button to see:
- Sound descriptions in bottom-right
- Direction arrows (→←↑↓)
- Works up to 128m away
- Smooth fade animations

### 👣 Footsteps

Enable "Footsteps" to see:
- Movement trails behind players/NPCs
- Color-coded by faction
- Fades out after 3 seconds

### 🎯 Aimbot

| Button | Color | Use With |
|--------|-------|----------|
| GunBot | Red | Pistols, rifles, shotguns |
| MeleeBot | Orange | Crowbars, fire axes, etc. |

> **Note**: Must be in combat mode (left-click) to work

## Troubleshooting

### Features not working?
- Ensure mod is loaded (check console)
- Some features require being in-game (not in lobby)

### Performance issues?
- Disable unused overlays
- Features update less frequently when stationary

### Sound subtitles not showing?
- Enable "Sounds" button
- Must be within 128m of sound source

---

## Tips

1. **Compass**: Best used with coordinate tracking
2. **Sound Subtitles**: Great for hearing impaired or low volume
3. **Footsteps**: Useful for tracking escaping antagonists
4. **Aimbot**: Use responsibly - may affect fair play
